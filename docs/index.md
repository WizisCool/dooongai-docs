# DooongAI API 文档

欢迎使用 `DooongAI`。

DooongAI 是公益性质的一站式 AI 模型 API 中转服务，提供 OpenAI 兼容接口，帮助你用熟悉的 SDK 与调用方式快速接入。

## 文档导航

- `快速开始`：注册、创建 API Key、首个请求验证
- `API 鉴权`：请求头规范与密钥安全
- `Base URL 与模型`：地址规范、模型选择与上线前检查
- `API 接口`：模型列表、对话请求、流式输出、错误处理

## 推荐阅读顺序

1. 快速开始
2. API 鉴权
3. Base URL 与模型
4. API 接口总览
5. API 请求示例

## 兼容性说明

默认使用 OpenAI 兼容协议。若你已接入 OpenAI SDK，通常只需替换：

- `baseURL` -> `https://ai.dooo.ng/v1`
- `apiKey` -> 你的 DooongAI API Key
